# Experiments

This directory is the run ledger for the TinyAya TTS Lab.

Use it for:
- structured run indexes under `experiments/runs/`
- human-readable experiment summaries and snapshots
- operator-facing protocol docs

Interpret the layout like this:
- `recipes/`
  - canonical run definitions
- `experiments/`
  - what actually ran
- `research/`
  - why it ran and how it was judged

Historical logs and W&B links stay here even if they still mention older
project names. Do not rewrite them unless there is a concrete migration reason.
